# Software-Design-Lab5
Лабораторная работа по Конструированию программного обеспечения

## Описание
В приложении реализован обмен данных между приложениями тремя способами: 
- Буфер обмена Windows;
- Отправка сообщений WM_COPYDATA SendMessage;
- Сокет System.Net.Sockets: TcpClient, NetworkStream, TcpListener.

Форматы данных: текст, изображение, и структура, состоящая из двух чисел и индекса операции, соответствующего одной из простых математиче-ских операций: +, -, *.

Приложение Client отправляет данные приложению Server. Приложение Server получает данные от «Клиента» и обрабатывает их простыми методами (для текста – приведение к верхнему регистру, для изображения – поворот и отражение).

Консольное приложение Calculator получает входные данные (два числа и операцию) в качестве аргументов командной строки, выполняет вычисление и записывает протоколы об операциях в таблицу БД запросом INSERT. Client отправляет Server команду на запуск Calculator через SendMessage, при получении команды Server запускает Calculator, передавая в качестве аргументов командной строки данные структуры, ранее полученные от Client.
